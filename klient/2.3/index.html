Synlighet

Element kan döljas på olika sett,se nedan för skillnaden mellan visibility och display. Man kan även hantera när innehållet
blir för stort för elementets box med overflow med värde för önskat beteende. Stora bilder kan beskäras med clip, så att en mindre fyrkant av bilden visas.

Visibility
Dold/visas. Med visibility: hidden; tar elementet upp plats. Se även Display
Display
Dold/visas. Med display:none; är elementet helt osynligt och tar ej upp plats
Clip
Om en bild är större än sitt element kan man beskära den genom att ange 4 koordinater som värde
Overflow
Styr hur innehåll som flödar över ett elements box ska behandlas.

Mer effekter

Nu ska vi skapa en box som till viss grad är transparant, har rundade hörn, multipla bakgrundsbilder, skuggor och gradienter.
Transparans sätt med opacity och ett värde mellan 0 och 1. Border-radius sköter rundade hörn och värdet sätts i procent(&).
Box-shadow ger boxen skuggor. Gradient kan antingen vara av typen radial-gradient och sätts med minst två färger men kan även ha fler, exempel:
background: radial-gradient(red, yellow, green); (färgerna anges innifrån och ut).
Exempel med linear gradient istället, och 45 graders vinkel: background: linear-gradient(-45deg,#ac2925, yellow);
