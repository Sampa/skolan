Tabeller

Table-layout
Ett css attribut, kan sättas till auto eller fixed.Auto sätter columnbrädden utifrån det bredaste, ej delbara, innehållet. Det gör att hela tabellen måste läsas igenom innan den kan börja visas. Fixed bryr sig inte om innehållet, utan tabellens bredd och bredden på kolumnerna och kan börja visas när första raden tagigts emot.
Border-collapse
Styr hurvida linjerna hålls seperata eller slås samman till en gränslinje. Exemplet med fixed tabell har kolappsade gränser, medans auto tabellen har sepererade gränslinjer.
Border-spacing
Sätter avtåndet mellan linjerna ifall border-collapse är satt till seperate. Exempel, 5px horizontalt och 10px vertikalt avstånd: 
border-spacing:5px 10px;
Caption-side
En tabell kan ha en caption, vars position (top/bottom) kan sättas med detta attribut.
Empty-cells
Med border-spacing:seperate; kan man med empty-cells:hide; dölja tomma celler. Standard är att de visas.

Exempel

table-layout:auto;
1000000000000000000000000000 10000000
table-layout:fixed;
1000000000000000000000000000 10000000

CSS med formulär

Ett formulär kan stilsättas på många olika sätt. Ett tips är att utnyttja de olika taggarna och ge dem stilattribut. Det kan även vara användbart att gruppera ihop taggen med dess formulärfält i ett omslutande element för att underlätta stilsättning och positionering.

Exempel

Formulär formaterat med CSS 

            <form>
                <fieldset>
                    <legend>Formulär formaterat med CSS</legend>
                    <section>
                        <label for="name">Name</label>
                        <input type="text" id="name">
                    </section>
                    <section>
                        <label for="man">Man</label>
                        <input type="radio" id="man" name="sex">
                        <label for="woman">Kvinna</label>
                        <input type="radio" id="woman" name="sex">
                        <label for="country">Land</label>
                    </section>
                    <section>
                        <label for="country">Land</label>
                        <select id="country">
                            <option value="sweden">Sverige</option>
                            <option value="finland">Finland</option>
                        </select>
                    </section>
                    <section>
                         <label for="about">Om dig</label>
                        <textarea id="about"></textarea>
                    </section>
                </fieldset>
            </form>
