Grundläggande Javascript

Externt länkad JavaScript-fil och dess fördelar

Javascript kod bör skrivas i ett eget dokument med filändelsen .js, du inkluderar sedan filen i ditt html-dokument. Exempel: 

<script src="minjavascript.js"></script>
(koden skrivs i dokumentets <head> tag).
Ibland inkluderar man javascript filerna i slutet av dokumentet, då laddas de in efter övrig kod körts vilket gör att din sida upplevs snabbare för besökarna.

Du kan såklart ha flera olika javascript filer i samma dokument, och inkludera samma fil på flera olika html-sidor. Din kod blir lättare att dela med sig av eller återanvända. Du får även tydligare/renare och mer användbar kod när olika språk hålls seperat.

Datatyper och variabler

I javascript behöver du inte ange datatyp för dina variabler. Men tänk på att koden exekveras från vänster till höger och därmed kan ge olika resultat, så var noga med att se till att svaret blir det du förväntar dig.
Du kan använda både ' och " för att omsluta en sträng. Innehåller en sträng det ena tecknet, använd det andra runtom. För att tilldela värden i Javascript används operatorn =. Exempel på hur du skapar variabler och tilldelar : 


            var foo = 10+10+"foo"; //Sträng, foo= 20foo
            foo = 'foo'+10+10; //Sträng, foo är nu = foo1010
            foo = "It's funny"; //Enkelfnutt innanför dubbelfnuttar
            foo = 'He said "hey man"'; //tvärtom från ovan
            var ett = 1; //Int
            //eller skapa fler på en gång
            var etta, tva, tre; //okänd datatyp
            etta = 1;
            tva = 2;
            tre = 3;
            var summa = etta+tva+tre; //Int, Summa=6

Array, en datastruktur

Arrayer i javascript är nollbaserad, dvs första värdet finns på index (plats) 0, det andra på plats 1 osv.
En array skapas såhär: 


            //variant 1
            var myArray = ["a","b",c"];
            //variant 2
            var myArray;
            myArray[0] = "a"; //index 0
            myArray[1] = "b"; //index 1
            myArray[2] = "c"; //index 2
            myArray[3] = "d"; //index 3
            //variant 3
            var myArray = new Array("a","b","c","d");
För att komma åt värden i en array är refererar du till dess index: 

            //genererar en popup med värdet på index 0 = "a"

En sträng kan ses som en array av bokstäver som tillsammans bildar text. Därför går det enkelt att få reda på vilken bokstav som döljer sig på en viss position 


        //Skapa en sträng
        var foo = "Hej"; //eller 'Hej'
        var first = foo[0]; //first = H
        //med citat i strängen
        var quote = "En person sade \"Vi hade iallafall tur med vädret\""; // \ krävs för att ett citat tecken ska tolkas som vanlig text
        var quote2 = 'En person sade "Vi hade iallafall tur med vädret"';// med enkelfnuttar runtom behövs inget \
        //Sätta ihop flera olika strängar med hjälp av + (Konkatenera)
        var all = foo+", första bokstaven är: " + first + ". Och här är " + 1 + " citat " + quote;

Det finns även en mängd funktioner som går att användas på strängar. Mer om funktioner senare i den här tutorialen, men här är ett enkelt exempel på hur man hittar vilken position en bokstav har. 


        var foo = "Hej";

Aritmetrik

Addition
För att addera 2 med 1 kan du välja på: 2+1, 2++ samt ++2. "++" adderar alltid med 1. Du kan även addera strängar: 

                var foo= "Hatt";
                var bar = "ar";
                var fooBar= foo+bar; //fooBar = Hattar
                var mix = foo+1+"tur";//mix = hatt1turq§
Subtraktion
För att subtrahera 2 med 1 kan du välja på: 2-1, 2-- samt --2. "--" subtraherar alltid med 1
Multiplera 2 med 3: 2*3
Dividera 6 med 3: 6/2
Modulo: 67%10

Jämförelse

  • Är dem lika? foo==fooBar
  • Är dem olika? foo!=fooBar
  • Är foo minst? foo < fooBar
  • Är foo mindre eller lika med fooBar? foo <= fooBar
  • Är fooBar minst? foo > fooBar
  • Är fooBar mindre eller lika med foo? foo >= fooBar

Flödeshantering (if-satser och loopar)

If-satser ("antingen eller") används för att köra olika satser beroende på ett villkor. 


        //Exempel som alltid kommer att köras (1 kommer ju alltid vara mindre än 2):
        if(1<2){
            console.log("1 är mindre än 2)";
        }
        //Exempel som skapar en console.log med olika text beroende på en variabels värde
        var foo = 11;
        if(foo<10){
            console.log("foo är mindre än 10");
        }else{
            console.log("foo är större eller lika med 10");
        }
En loop består av ett villkor och en sats. Satsen körs sedan sålänge villkoret är true. Därför är det viktigt att undvika oändliga loopar. Det går att göra loopar på tre sätt:
While-loop
Den enklaste loopen att lära sig och klarar av det mesta. Obs! Kom ihåg att använder du ++ för att öka värdet på en variabel ska du tänka på att 

                    ++foo = foo ökas med ett innan det bearbetas
                    foo++ = foo ökas först efter att det bearbetats. Det kan ge olika resultat i vissa fall.
                    //While loop
                    var a= 0; //Vi behöver en variabel för att villkoret ska kunna gå från true till false
                    while(a < 10){
                        console.log("A är "+a++);
                        //varianter på ovan
                        a = a + 1;
                        console.log("A är "+a);
                        //få loopen att skriva ut 0 först istället för 1
                        console.log("A är "+a);
                        a++; //eller ++a, a+1;
                    }
For-loop

                    for(x=0;x< 10; x++){
                        console.log("X är" + x);
                    }
Foreach-loop
Foreach används för att loopa igenom en array eller lista med objekt. Den körs en gång för varje värde. I vårt exempel anropas en funktion varje varv, vilket kan göra din kod snyggare och lättare att dela upp. Det går även bra att återanvända samma funktion i olika loopar. Mer om funktioner kan du läsa under nästa rubrik. 

                    var nummer = ["a","b","c","d"];
                    nummer.forEach(fooFunc());
                    function fooFunc(item,index){
                        console.log("item = " item + " index = " + index);
                    }

Funktioner

En funktion defineras med function följt av ett valfritt namn på funktionen som sen används för att köra koden i funktionskroppen. En funktion kan även ha parametrar. En, flera eller ingen. En parameter fungerar som en variabel, men enbart inom funktionen. Försöker du komma åt värdet utanför måsvingarna kommer du få ett error. 


        //funktionen tar emot två parametrar och skapar en sträng av dem.
        function myFunkyFunc(foo,bar){
            return "foo = " + foo + ", bar = " + bar;
        }
        //anropa, sätter x till "foo = hej, bar = 341"
        var x = myFunkyFunc("hej", 341);
        //ett html element kan användas för att anropa funktioner i Javascript

Reguljära uttryck

Ett reguljärt uttryck är ett mönster som körs mot en text för att hitta de instanser som passar in på mönstret (uppfyller alla "krav" i uttrycket). Varje reguljärt uttryck består av en serie tecken och kan användas till allt från att titta ifall en sträng innehåller ett särskiljt ord, till mycket mer avancerade exempel. Reguljära uttryck kan ta tid att få till rätt, och är väldigt svåra att komma ihåg. Därför är det en bra idé att ha ett litet "cheatsheet" där du samlar dina reguljära uttryck och vad dem gör. När du kört ett reguljärt uttryck skapat ett objekt med information om resultatet. 


            var foo = "jag Jag jag";
            //Se ifall en sträng innehåller ordet "jag".
            var match = foo.match(/Jag/); //ger resultatet "Jag", då det är case-sensitiv
            var match = foo.match(/Jag/i); //ger resultatet "jag", tack vare modifieraren i. Ej case-sensitiv
            var match = foo.match(/Jag/ig); //ger resultatet "jag, Jag, jag", då modifieraren g hittar alla instanser, inte bara den första
