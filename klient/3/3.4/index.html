Gör ett eller flera JavaScript-program som använder följande webbläsar-objekt: Window: Egenskaper: navigator, screen, location screenLeft, screenTop, screenX, screenY innerHeight, innerWidth, outerHeight, outerWidth pageXOffset, pageYOffset Metoder: alert, confirm, prompt open, close focus, blur setInterval, clearInterval, setTimeout, clearTimeout moveBy, moveTo resizeBy, resizeTo scrollBy, scrollTo print Navigator: Egenskaper: appCodeName, appName, appVersion, platform, userAgent cookieEnabled Screen: Egenskaper: width, height, availWidth, availHeight colorDepth, pixelDepth Location: Egenskaper: host, hostname, port href, protocol, pathname, hash, search Metoder: assign reload replace HTML-dokumenten ska följa standarden HTML5 och varje JavaScript-fil ska vara extern. Testa att allt validerar med W3C:s validator.
