Info från localstorage

När du trycker på töm local storage töms det för båda sidorna. Men du kan ångra det eftersom det lagrade objektet finns som backup i session storage.

Om du ångrar raderingen sparas information återigen i local storage och formuläret på index.html fylls igen.

Klickar du på töm session storage tas backup-objektet bort och det går det inte längre att ångra tömningen från local storage.
