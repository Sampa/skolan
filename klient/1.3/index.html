Mark-taggen

Mark taggen används för att markera en del av en text

Time med datetime

Time används runt ett tid/datum i ett format för människor, exempel: <time>10:00</time> vilket ger: 10:00.
Tiden/datumet ska vara i ett giltigt format som de är definerade på w3.org

Time med datetime attribut kan användas för att definera en tid/datum maskiner kan läsa.
<time datetime="2016-12-24 15:00">Kalle Ankas Jul på tv!</time> ger:
Kalle Ankas Jul på tv!
Då kan det i time elementet vara valfri text. Datumet/tiden kan därmed översättas mellan olika format vilket är användbart då olika länder har olika format för att beskriva samma datum.
Andra fördelar är smartare sökresultat från sökmotorer eller att man kan schemalägga händelser för en användare utan att veta vilket datumformat hen använder.

Wbr

Wbr taggen används för att "upplysa" webbläsaren om vart det är lämpligt att bryta en text med ny rad OM det behövs, annars visas hela texten på en rad.
Prova att ändra storleken på webbläsarfönstret för att se effekten i texten nedan som har en wbr tag vid varje kommatecken,punkt eller frågetecken. Texten är sammanhängande utan <br/>, eller mellanslag.
