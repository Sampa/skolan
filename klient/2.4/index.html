CSS och img-taggen

Beskriv och exemplifiera även hur man använder CSS med img-taggen. Visa hur man gör så att en text flödar (med ett visst avstånd) bredvid en bild så att det inte blir (för stort) tomrum bredvid bilden men om det skulle råka vara en rubrik (h2, h2 och så vidare) i denna text så ska allt (inklusive rubriken) fortsätta under bilden (se egenskapen clear).

Clear

Clear styr på vilka sidor ett flytande element får vara. Därför kan det användas tillsammanns med float för att t.ex placera text bredvid en bild.

Exempel:

Eftersom vi vill ha bilden till vänster placerar vi ut den i HTML-koden och sätter float:left; på den. Vi sätter även en högermarginal på 10px för att skapa lite mellanrum till innehållet åt höger, i vårt fall texten. Sen sätter vi en regel som gör att alla rubriker i texten bryter detta med clear:both; 


                    #bildtext1 h1,h2,h3,h4,h5,h6{
                        clear: both;
                    }

                    #jennie{
                        float: left;
                        margin-right: 10px;
                    }

                    <section id="bildtext1">

För att begränsa bredden på texten och samtidigt ha den kvar bredvid bilden lägger vi till:


                /* första p taggen i #bildtext1 elementet */
                #bildtext1 p:first-of-type{
                    max-width: 300px;
                    float:left;
                }
