Externt länkad CSS-fil

CSS-kod används framförallt till att stilsätta html dokument med färger, typsnitt och placering av element.
För att kunna använda sin css kod på mer än ett element skriver man koden i en extern css fil som har filändelsen .css.
Filen inkkluderar man sen i de html dokument man vill stilsätta. Med en extern CSS mall kan man alltså stilsätta
mer än ett html dokument. CSS är uppbyggt av enskiljda regler, som kan utläsas som t.ex "texten ska vara röd".

Man kan sen gruppera ihop flertalet regler med varandra som man sen applicerar på ett enskiljt element med hjälp av selektorer.
Koden appliceras då på alla element selektorn matchar. En selektor kan bland annat vara ett elements id, namn eller en html tag.
Om man döper sin fil till style.css skriver man <link rel="stylesheet" href="style.css">
och placerar raden inom html dokumentet/dokumentens <head> tag. Ligger css filen inte i samma mapp som html dokumentet
så får man anpassa sökvägen i href attributet.

Arv

Arv (En. inheritance) innebär att en regel ärvs av element inuti det man sätter sin CSS-regel på.
I exemplet nedan blir både text1 och text2 påverkade av regler som sätts på div elementet. 


            //css
            div {
            color: red;
            }
            //html
            <div>
            Text1
            <p>
            Text2
            </p>
            </div>

Cascade

Cascade tekniken innebär att ett elements slutgiltiga stilmall kan innehålla regler från flera olika ställen samtidigt:

  • Webbläsarens inbyggda stilregler
  • Regler satta av användaren
  • Regler satta av dokumentents skapare:
    • I ett externt css dokument
    • Direkt i ett html dokument
    • Direkt i ett html element

Det tillsammans md arv gör koden flexibel och återanvändbar men även svårare att felsöka.

Selektorer

Selektorer använder man för att wälja ut de element ens regler ska appliceras på. Syntax:


            Selektor{
            regelnamn : värde;
            regel2 : värde2;
            }
Tagg
Väljer alla element med motsvarande tag. Exempel: div, p
Attribut
Väljer element som har ett visst attribut. Exempel: div[id]. Man kan även välja utifrån attributens värden.
Klass
Väljer element utifrån klassnamn. Exempel: .minClass
ID
Väljer elementet med det specifika id:t. Exempel #fooid
Barn
Med barn selektorn kan man välja element som ligger direkt under något annat.
Exempel: article > header. 

            <style>
                .minClass {
                    font-size: 120%;
                }
                #fooid {
                    color: red;
                }
                h3{
                    margin-left: 30px;
                }
                article > header {
                    padding: 10px;
                }
            </style>
            <article class="minClass" id="fooid">
                <header>
                    <h3>Exempel</h3>
                </header>
            </article>

Enheter absoluta

Saker mätt i absoluta enheter visas i exakt den storleken. Rekommenderas inte på skärmar då de finns
i så varierande storlekar utan passar bättre till t.ex skriv-layouter där man vet storleken på enheten.

In
Tum (1in = 1 inch = 2.54cm)
Cm
Centimeter
Mm
Millimeter
Pt
Punkter (1pt = 1 points = 1/72 inch)
Pc
Picas (1pc = 12pt)

Enheter relativa

Relativa storlekar anger en längd i relation till ett annat längd värde.
Detta ger bättre skalning på innehållet mellan olika medium.

Em
Relativt till textstorleken på elementets nuvarande font (2em = 2 gånger så stor )
Ex
Relativt till x-höjden på fonten
Px
Pixel relativa till skärmen. 1px är antingen 1 enhetspixel (på lågupplösta skärmar) medans det på skrivare eller skärmar med högupplösning
kan betyda flera enhetspixlar.
%
Procent av det omgivande elementet

Ett CSS dokuments uppbyggnad

Se selektorer för exempel på syntax för att välja ut html element och
> extern CSS-fil för hur man länkar till en css fil från sitt HTML-dokument.

Exempel med flera selektorer


            selector1, selector2, selector3 {property: value; property: value;}
            div, .news, #foo {color:yellow; margin-left: 30em;}

Man kan importera andra stilmallar med @import url("stylesheet.css");
Dokumentet är alltså uppbyggt av regler sammansatta till block med selektorer. Kommentarer sker med /* kommentar här */. Ordningen på reglerna spelar ingen roll. Däremot har de en viss hierarki som avgör vilken regel som appliceras om den finns
i flera block med selektorer som matchar ett givet element.

Hierarki för att välja regel om det i ett elements slutgiltiga stilmall (se cascading) finns dubletter.

  1. Inline, regler skrivna i elementets style attribut
  2. ID, regler skrivna i block med en id (#) selektor
  3. Klasser, attribute och pseudo-classer (:hover, :focus osv)
  4. Element och pseudo-element (:after, :before osv)
